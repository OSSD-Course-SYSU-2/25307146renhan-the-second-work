# 雪怪视觉小说游戏

## 版本

当前版本：1.0.0

## 简介

一个基于HarmonyOS的视觉小说游戏，你可以自定义虚拟人物和剧情，软件会自动播放你的人物和剧情，就像会翻面的连环画。

## 功能特性

- 剧情自动播放（4秒倒计时跳转）
- 视觉展示（背景图、立绘、对话框）
- 分支选择系统（用户交互影响剧情走向）
- 多屏适配（手机、平板）
- 双结局分支（给玉米/不给玉米）
- 重新开始功能

## 技术栈

- HarmonyOS ArkTS
- 响应式布局（百分比+fp单位）

## 运行环境

- HarmonyOS 6.0.2(22) 及以上
- 支持设备：手机、平板
